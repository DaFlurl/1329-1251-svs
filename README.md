# GitStyle Webseite

Moderne Webseite mit Git-basiertem Hosting.

## Struktur
- `index.html` - Hauptseite
- `style.css` - Stylesheet
- `README.md` - Dokumentation

## Hosting
- GitHub Pages
- GitLab Pages
- Netlify
- Vercel

## Installation
```bash
git clone <repository-url>
cd gitsitestylewebsite
```

## Deployment
1. Push zu GitHub
2. GitHub Pages aktivieren
3. Fertig!

# GitHub Pages Configuration

## Build Settings
- Source: Deploy from a branch
- Branch: main  
- Folder: / (root)

## Static Files
All HTML, CSS, JS, and JSON files should be served as static content.

## No Jekyll Processing
The .nojekyll file disables Jekyll processing for better compatibility.

## Files Available
- index.html (main entry point)
- dashboard.html (unified dashboard)
- scoreboard.html (main scoreboard)
- history_simple.html (simple history view)
- history_dashboard.html (advanced history view)
- monday_data.json (current data)
- scoreboard-data.json (historical data)

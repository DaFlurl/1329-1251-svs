# VS Code Docker Commands
# Strg+Shift+P → Befehl eingeben

## Container Management
Docker: Compose Up           # Startet alle Services
Docker: Compose Down         # Stoppt alle Services  
Docker: Compose Restart      # Startet Services neu
Docker: Show Logs           # Zeigt Container Logs

## Dev Container
Dev Containers: Reopen in Container  # Öffnet Projekt im Container
Dev Containers: Attach to Running Container

## Images
Docker: Build Image         # Baut Docker Image
Docker: Push Image         # Pushed zu Registry
